查看是否加密=脱壳

  • 未加密
    ➜  Aweme.app otool -l Aweme | grep crypt
    cryptoff 28672
    cryptsize 4096
      cryptid 0
    • 说明
      • cryptid 0：表示未加密 = 已脱壳
  • 已加密
    ➜  Aweme.app pwd
xxx/Aweme抖音/iPhone7-137black/Aweme.app
➜  Aweme.app otool -l Aweme | grep crypt
    cryptoff 28672
    cryptsize 4096
      cryptid 1
    • 说明
      • cryptid 1：表示已加密 = 未脱壳
