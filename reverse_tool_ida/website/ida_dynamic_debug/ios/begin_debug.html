开始调试

开始调试

调试命令

  • IDA基本调试命令
    • Step Into = F7 = 单步进入 = 单步步入
    • Step Over = F8 = 单步执行 = 单步步过
    • Run until return = Ctrl+F7 = 运行直到返回
    • Run to cursor = F4 = 运行到鼠标处
    • Run until message received = Alt+O

调试菜单

顺带看看相关菜单：

  • IDA->Debugger
    • Debugger window
    • Breakpoints
    • Watches
    • Tracing
