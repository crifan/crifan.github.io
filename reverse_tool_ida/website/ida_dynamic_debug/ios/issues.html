IDA调试iOS

每次调试启动太耗时

  • IDA调试iOS的app：WhatsApp
    • 每次调试，都会
      • Importing Symbols导入符号：耗时10分钟左右
      • 自动分析：耗时很久

-》导致每次调试都要等待很长时间：平均要半小时以上

-》导致IDA调试iOS的app：效率太低，不方便，几乎不可用

无法类似lldb的po打印出对象信息

此处IDA调试iOS的app：WhatsApp

0x108E87290
