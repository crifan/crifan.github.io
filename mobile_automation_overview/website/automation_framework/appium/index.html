Appium

Appium

  • 概述
    • Appium是最主流的自动化测试框架，支持自动化操作iOS手机、Android手机和Windows系统中的原生、移动Web和混合的应用。
      • 原生应用：用iOS、Android或Windows SDKs编写的应用
      • 移动Web应用：用移动端浏览器访问的应用
        • iOS上的Safari、Chrome
        • Android上的内置浏览器
      • 混合应用：带有一个Webview的包装器，用来和Web内容交互的原生控件
