getter和setter

类的属性的getter和setter

之前接触计算机语言，比如C、C++等，都没有getter和setter的

后来接触到了一些语言：

  • C#
  • Swift
  • Python

其对象==类==Class中都有了属性Property的：

  • getter：读取属性的值
  • setter：设置（写入）属性的值
