画图工具概述

对于画图工具来说：

  • 图有多种类型
    • 常见的=通用的
      • 图表类
        • 饼图
        • 柱状图
        • 等
      • 图chart类
        • 流程图=flow chart
        • 脑图=mindmap=思维导图
        • 其他
          • 和地图相关的
  • 有多种画图的工具
    • 使用方式
      • 本地客户端软件
        • Win
        • Mac
          • Diagrams
        • Linux
      • Web网站=网页端
      • JS库
        • 独立JS库
      • 特殊格式
        • Markdown
    • 按图的类型分
      • 通用图形
        • 流程图
      • 专门某大类
        • 脑图
          • 客户端软件
            • Xmind
          • 在线网站
            • processon
        • 地图
          • JS库
            • leaflet.js
              • Python库
                • Folium
