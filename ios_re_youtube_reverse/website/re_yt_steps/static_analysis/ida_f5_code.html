IDA分析代码逻辑

IDA分析代码逻辑

主要涉及，分别用IDA加载和分析，YouTube的2个二进制的文件：

  • YouTube
  • Module_Framework

然后分析其代码逻辑。

其中最核心用法是，找到可以的要研究的类的函数后，去Functions窗口中，通过函数名搜索函数，找到对应汇编代码，再F5查看反编译后的伪代码，尝试搞懂代码的具体逻辑。
