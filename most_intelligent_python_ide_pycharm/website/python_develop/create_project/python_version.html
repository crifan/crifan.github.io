设置Python版本

设置Python版本

新建Python项目后，记得要

  • 选择Python版本
    • =设置Python环境
    • =设置Python解析器
    • =Python虚拟环境
      • （如果是虚拟环境的话）
  • 尤其是
    • 当系统中安装多个版本的Python
      • 既有Python 2
      • 又有Python 3
        • 甚至有多个版本的Python 3
    • 或本身有个Python虚拟环境
      • 比如用virtualenv/pipenv等创建出来的Python虚拟环境

否则会出现：

  • 无法正常调试
    • 找不到已安装的库

下面详细介绍一下，新建Python项目后后，如何设置对应的：

PyCharm中设置当前项目的Python解析器
