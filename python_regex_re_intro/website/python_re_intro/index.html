re是Python中内置的正则表达式模块。功能十分强大。

先概述如下：

  • 最常用：
    • 搜索：re.search
    • 替换：re.sub
    • 匹配：re.match
  • 其他
    • 匹配所有：re.findall
    • 匹配所有，且每个都可获取匹配对象的详情：re.finditer
      • = re.findall + re.search
