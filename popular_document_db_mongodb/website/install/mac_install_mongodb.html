Mac中安装MongoDB

Mac中安装MongoDB

接下来详细介绍Mac中如何安装MongoDB

目前免费的Community社区版MongoDB最新版是：v4.2

Mac中安装最新版MongoDB 4.2 community社区版方式：

先去：

brew tap mongodb/brew

再去安装MongoDB：

brew install mongodb-community

再去[可选]安装mongo shell：

brew install mongodb-community-shell
