Xcode

Xcode

苹果的平台的软件的IDE：Xcode

可用来开发苹果旗下的众多平台的软件：

  • 桌面端
    • macOS
  • 移动端
    • iOS
    • watchOS
    • tvOS

优点

集成了和自己的语言和环境高度整合的功能和智能提示
