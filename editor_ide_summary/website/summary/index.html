编辑器和IDE总结

编辑器和IDE总结

  • 总体推荐逻辑：开发不同的技术，推荐用对应合适的编辑器
  • 具体推荐哪款：
    • 命令行编辑文件：vim
      • 尽量用vim，比vi更好用，往往有代码高亮，效果更好
    • Android：Android Studio
    • Swift/OC：Xcode
    • Python
      • 重量级：PyCharm
      • 轻量级：VSCode+Python插件
    • Web端
      • 小程序：微信开发者工具
        • 也可偶尔配合用VSCode编辑代码
      • HTML+CSS+JS
        • 重量级：WebStrom
        • 轻量级：VSCode
    • 桌面端
      • Windows
        • C#：Visual Studio
    • 编辑其他各种文件：VSCode
