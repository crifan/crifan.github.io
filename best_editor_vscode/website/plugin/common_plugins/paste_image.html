Paste Image

Paste Image

先去插件中安装插件：Paste Image

然后确保当前系统剪贴板中有图片，比如复制了图片或者截了图

然后Mac中用快捷键Command+Alt+V，稍等一下，即可粘贴出图片到当前目录，且在当前Markdown中插入图片的内容了：

![](2020-02-15-22-24-26.png)

效果如图：

如此，方便的在当前Markdown同目录下插入图片，效果很不错。
