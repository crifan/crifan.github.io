debugserver+lldb调试

此处介绍用debugserver+lldb去调试iOS程序。

此处举例说明：

  • iPhone机型：iPhone 7 Plus, iOS 13.4.1
  • 被调试app：抖音
  • 调试方式：debugserver+lldb的命令行

核心思路是：

  • iPhone中：确保debugserver有正确的权限entitlement
    • 先从iPhone中导出debugserver
    • 再去Mac中，（用codesign或ldid）给debugserver加上合适的权限
    • 再把加了entitlement的debugserver拷贝回iPhone中
  • iPhone中：用debugserver启动(抖音)app
  • Mac中：用lldb去调试(抖音)app
