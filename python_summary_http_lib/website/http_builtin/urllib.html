urllib

urllib

此处先介绍Python的内置网络库的发展历史：

  • 背景：Python有2个大的版本：Python2和Python3
  • Python2
    • 有独立的2个库：urllib 和urllib2
      • 缺点：功能划分的不是很清楚
  • Python3
    • 所以合并了urllib和urllib2为单个库：urllib
      • 其中：Python2的urllib2合并成Python3的urllib.request
