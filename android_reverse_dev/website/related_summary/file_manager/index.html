安卓逆向期间，常涉及到文件管理方面的内容。

文件上传和下载=导入导出文件

  • 用文件管理器
    • 文件极客
  • 用命令：adb
    • 概述
      • （从安卓中）导出文件
        adb pull <remote_android> [<local_pc>]
      • 导入文件（到安卓中）
        adb push <local_pc> <remote_android>
    • 详解
      • 安卓自动化工具：adb

文件目录的关系

常见目录：

  • /sdcard：SD卡根目录，即普通用户保存文件的根目录
    • /sdcard/Download：下载目录
      • 安卓手机中，通过浏览器等工具下载的文件，往往默认保存在到下载目录，就是这个：/sdcard/Download
