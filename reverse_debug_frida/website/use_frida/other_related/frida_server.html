frida-server

frida-server

越狱手机iPhone安装Frida后，就可以通过Frida插件的已安装文件：

看到安装了2个核心文件：

  • /usr/lib/frida/frida-agent.dylib
  • /usr/sbin/frida-server
    • 其中frida-server就是Frida的server端
