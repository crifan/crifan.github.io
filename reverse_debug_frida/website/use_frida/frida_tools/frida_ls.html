frida-ls

frida-ls

  • frida-ls
    • 是frida-tools中一个工具
    • 作用：显示系统中的文件列表
      • 类似于ls
      • 支持：当前（Mac等）PC端的文件和（iPhone等）移动端中的文件

用法

  • 显示当前PC端的文件：不加参数
    frida-ls
  • 显示USB连接的（iPhone等）移动端中的所有文件：-U == USB
    frida-ls -U
