V2Ray

V2Ray可视为ss的下一代技术。

其基本用法和思路，和ss一致，不多赘述。

  • 总体思路
    • 服务端
      • 自己搭建服务器或买第三方的服务
    • 客户端
      • 再去使用某个V2Ray的client端
      • 加上配置
        • 即可科学上网
