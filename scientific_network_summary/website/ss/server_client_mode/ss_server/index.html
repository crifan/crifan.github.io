Shadowsocks的服务器

想要用Shadowsocks的客户端+服务器模式去科学上网的话，需要先有ss服务器。

拥有ss服务器的方式有几种：

  • 从第三方服务商那里购买ss相关服务
    • 最新推荐：购买SSR服务，比SS更稳定，不易被封
  • 自己搭建ss服务器
