预览和发布API文档

下面介绍Postman中如何预览和发布API文档。

简要概述步骤

  1. Collection
  2. 鼠标移动到某个Collection，点击 三个点
  3. Publish Docs
  4. Publish
  5. 得到Public URL
  6. 别人打开这个Public URL，即可查看API文档

详细解释具体操作

预览API文档

点击分组右边的大于号>

如果只是预览，比如后台开发员自己查看API文档的话，可以选择：View in web

等价于点击Publish Docs去发布：

View in Web后，有 Publish的选项（见后面的截图）
