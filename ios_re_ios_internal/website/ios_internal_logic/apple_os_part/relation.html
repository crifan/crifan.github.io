关系总结

此次给Apple操作系统常见概念的关系：

  • iOS
  • macOS
  • Darwin
  • XNU
  • Mach
  • BSD/FreeBSD
  • IOKit

做个总结：

  • Apple的不同平台有不同操作系统：iPhone的iOS、Mac的macOS等等
    • iOS、macOS等系统的内核，都是：Darwin
      • Darwin = XNU + 其他
        • XNU = Mach + BSD/FreeBSD + IOKit
  • 内核关系类比
    • Ubuntu
      • Ubuntu是平台(plaftorm)
        • Linux + GNU 是操作系统(OS)
          • Linux是内核(kernel)
    • macOS
      • macOS是平台(plaftorm)
        • Darwin 是操作系统(OS)
          • XNU是内核(kernel)
  • 概念范围
    • 文字
      • iOS > Darwin
      • macOS > Darwin
      • Darwin > XNU
      • XNU > Mach
      • XNU > BSD/FreeBSD
      • XNU > IOKit

最终总结

  • 苹果操作系统的架构 = Apple OS Architecture
