Xcode开发概览

Xcode开发概览

在弄苹果相关开发期间，往往要用到Apple苹果自带的这个IDE：Xcode

此处就整理相关的开发心得。

  • Xcode
    • 是什么：Apple苹果生态中，用来开发的IDE
      • 支持各种Apple的平台和系统
        • 桌面端：macOS
        • 移动端：
          • iPhone类：iOS
          • iPad类：ipadOS
          • Apple TV：tvOS
          • Apple Watch：watchOS
    • 功能
      • IDE的标准功能：编写代码、调试等
      • 额外：还内置附带了其他一些命令行工具
