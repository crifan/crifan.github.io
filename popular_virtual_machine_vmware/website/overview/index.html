概述

背景

想要在 Win10中用上macOS虚拟机

所以去调研：

【已解决】Win中用哪个虚拟机软件安装macOS比较好

结论

win中，适合安装macOS的虚拟机，主要是：

  • VM=VMWare
    • 常见版本：
      • VM 14
      • VM 15
        • 比如
          • VMware Workstation 15.1.0
  • 常见安装的macOS版本
    • macOS 10.14

最终决定：还是去用很多年之前就用过的：VMWare，去创建此处的macOS虚拟机。
