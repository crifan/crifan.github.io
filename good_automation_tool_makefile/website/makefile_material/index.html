官网资料

所有的资料中，最推荐的就是官网，解释的最权威和清楚。

不过是英文的，有能力的可以优先参考：

GNU make: Top

其中关于很多功能的解释，比较权威和清楚。

自己之前参考过官网的部分资料

比如之前就参考过：

  • 函数：GNU make: Functions
  • 想要输出信息，除了info，还有error和warning：GNU make: Make Control Functions
$(error text…)
$(warning text…)
$(info text…)
