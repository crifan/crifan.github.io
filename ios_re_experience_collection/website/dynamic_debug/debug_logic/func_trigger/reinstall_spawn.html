重装和spawn方式

重装和spawn方式

背景

研究iOS逆向期间，需要找某些iOS的ObjC函数的调用的时机和地方

以便于研究代码逻辑。

但是有时候，虽然找到并给函数写了hook代码，但是却没有触发

所以需要想办法，找函数被触发调用的时机

后来有了新思路：

  • 该函数很可能是：app初始化时，仅允许一次
    • 所以后续调试，无法再次触发
  • 如何触发
    • 模拟环境入手：重新安装app
      • 且用Spawn方式启动，模拟重新安装后首次启动
        • 或许就能触发
    • 从代码调用入手：借助于IDA的交叉引用xref，references to，继续向上找函数调用的地方
      • （虽然往往也很复杂，不好找，但）往往最后也是你能找到调用的地方的
