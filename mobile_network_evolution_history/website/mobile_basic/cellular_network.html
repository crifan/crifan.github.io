  • cellular system=（移动通信领域中的）蜂窝系统

    • 旧
      • 包含
        • GSM
        • UMTS
      • 特性：
        • 基于PS的GPRS/HSPA的方案
    • 新
      • 包含
        • LTE=Long Time Evolution
        • EPS=Evolved Packet System
      • 特性
        • 对于终端用户
          • 不提供任何传统的电信服务
            • 电话telephony
            • 视频电话video-telephony
            • 短信messaging
        • 而是基于：packet bearer services数据承载业务
          • 比如
            • 发送IP数据包
              • 且保证QoS质量
        • 通过IMS提供服务
          • 底层通过
            • LTE 访问网络 = LTE access network
