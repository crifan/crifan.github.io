加可调试权限再重签名

想要给单个的app=二进制，去实现可调试，可以：

  • 先加上可调试权限
    • 先导出原有entitlement权限
    • 加上相关的可调试权限
  • 再去重新签名=重签名
    • 重签名的工具
      • 推荐用：codesign
      • < iOS 15.0时也可以用：ldid
        • 详见：重签名 · iOS逆向调试：debugserver+lldb
