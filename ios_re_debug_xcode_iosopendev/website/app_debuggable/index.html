  • 什么是：app可调试debuggable
    • = 可调试debuggable
    • = （GUI的Xcode 和 命令行的debugserver）可以调试 = 有权限调试 任意的目标（app、二进制 的 进程）
      • 包括
        • Xcode可以调试任意app、二进制、进程
        • debugserver(+lldb)可以调试任意app、二进制、进程
