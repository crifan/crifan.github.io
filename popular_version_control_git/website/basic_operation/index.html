基本操作

Git基本操作

代码提交和同步代码

最常见的基本操作：

  • 概述
  • 详解
    • 最基本的
      • 创建本地git仓库
        git init
    • 最常见的三步：新增并上传文件
      • 添加文件
          git add
      • 提交
          git commit
      • 推送 = 上传到远端仓库
          git push
    • 更新文件
      • 2步
        • 下载新文件
            git fetch
        • 合并新文件
            git merge
      • 或：直接1步
        • 下载并合并
            git pull

代码撤销和撤销同步

相对高级一些的操作：

查看远程仓库url地址

git remote -v

其他操作

  • 其他
    • 常见操作
      • 暂存本地更改（往往在git pull之前）
        git stash
      • 恢复本地更改（往往在git pull之后）
        git stash pop
    • 高级操作
      • rebase
        git rebase
      • 分支
        git branch
git -b

格式：git_address#branch_name

npm install git://github.com/matthieuprat/react-tappable.git#fix-is-mounted-deprecation --save
