文件对比工具概览

文件对比工具概览

文件对比工具概述：

  • 免费
    • VSCode
    • WinMerge
    • Notepad++
      • 插件：Compare
    • diffuse
    • KDiff3
  • 收费
    • Beyond Compare
    • Meld

-》结论：

  • 优先推荐：Beyond Compare
    • 特点
      • 收费
      • 对比效果极好
        • 非常专业的内容对比软件
      • 支持格式众多
        • 除了普通文本类文件，还支持其他二进制类文件
          • 比如 音频类（Mp3等的头文件信息），word文档等等
      • 除了文件，也支持文件夹的对比
      • 支持对比时各种选项和模式
        • 比如 二进制快速比较 还是 基于规则的快速比较 等
  • 此次考虑：VSCode
    • 特点：
      • 免费
      • 文件内容对比功能还算够用
      • （目前）不支持文件夹级别的对比
        • 只支持单个文件互相对比
  • 再次考虑：WinMerge
    • 特点
      • 免费
      • 支持文件夹对比
